Estas métricas se calculan a partir de tus ejecuciones del simulador. Si tu navegador bloquea el almacenamiento local, los datos podrían no persistir.

Cómo lograr una puntuación sólida

La puntuación pondera Win Rate, R:R, PnL promedio, racha y el tamaño de muestra. El objetivo no es “100”, sino consistencia.

Muestra suficiente
Apunta a 30+ trades resueltos para mayor confiabilidad.
Constancia (racha)
Mantén periodos consecutivos con trades para mejorar el score.
Win Rate realista
Mantén un Win Rate estable (≈55%+) sin sobre-optimizar entradas.
R:R disciplinado
Evita mover SL/TP impulsivamente. R:R medio ≈1.2+ sostiene el score.
PnL promedio positivo
Busca PnL promedio neto positivo. Un 0.2%+ ya impulsa la puntuación.
Consejo: usa los filtros (Semana/Mes/Año) para detectar consistencia, no picos puntuales.
